# SDXL-Flux2VAE-ComfyUI-Node
Simple node to patch ComfyUI generation with the SDXL Flux2VAE arch support.  

## How to Use
Install the node pack. It patches SDXL Flux2 handling automatically while keeping Flux2/Klein behavior unchanged.
